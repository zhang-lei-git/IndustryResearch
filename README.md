# Industry Research

区域产业调研与机会决策工作台。

当前以西安市阎良区航空制造产业为首个试点区域，帮助数字化解决方案团队持续完成产业认知、企业调研、动态情报跟踪、共性需求归纳与市场/产品决策。

## 本地运行

```bash
npm install
npm run dev
```

浏览器访问 `http://localhost:3001`。

## 演进原则

阎良是首个区域，不是系统边界。后续通过“区域工作空间”扩展到多个区域；原始数据按区域隔离，行业知识、能力资产、调研模板和经治理的洞察可跨区域复用。
